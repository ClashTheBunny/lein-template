# shadow-cljs

[shadow-cljs](https://github.com/thheller/shadow-cljs) template.
Supports Om, Reagent and Rum.

## Usage

``` shell
lein new shadow-cljs your-project +reagent
lein new shadow-cljs your-project +om
lein new shadow-cljs your-project +rum
```
